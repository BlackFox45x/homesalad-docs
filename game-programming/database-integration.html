HomeSalad Database Integration

This page explains how the uMMORPG database is used as the backend for HomeSalad, with interactive diagrams showing the database structure and detailed explanations of how the database is integrated with the game systems.

Database Overview

HomeSalad uses the SQLite database system provided by uMMORPG Remastered as its backend. The database stores all persistent game data, including character information, inventory, quest progress, environment states, plants, buildings, and relationships.

The database schema has been extended from the base uMMORPG implementation to support HomeSalad's unique features, such as gardening, environmental transformation, and community relationships.

// Example of character data access in C#
public void SaveCharacter(Player player)
{
    using (var connection = new SqliteConnection(connectionString))
    {
        connection.Open();
        
        using (var command = connection.CreateCommand())
        {
            command.CommandText = @"
                UPDATE characters 
                SET level = @level, 
                    experience = @experience, 
                    health = @health, 
                    mana = @mana, 
                    strength = @strength, 
                    intelligence = @intelligence, 
                    dexterity = @dexterity, 
                    gardening_skill = @gardening_skill, 
                    building_skill = @building_skill, 
                    community_skill = @community_skill, 
                    last_position = @last_position, 
                    last_login = @last_login 
                WHERE name = @name";
                
            command.Parameters.AddWithValue("@name", player.name);
            command.Parameters.AddWithValue("@level", player.level);
            command.Parameters.AddWithValue("@experience", player.experience);
            command.Parameters.AddWithValue("@health", player.health);
            command.Parameters.AddWithValue("@mana", player.mana);
            command.Parameters.AddWithValue("@strength", player.strength);
            command.Parameters.AddWithValue("@intelligence", player.intelligence);
            command.Parameters.AddWithValue("@dexterity", player.dexterity);
            command.Parameters.AddWithValue("@gardening_skill", player.gardeningSkill);
            command.Parameters.AddWithValue("@building_skill", player.buildingSkill);
            command.Parameters.AddWithValue("@community_skill", player.communitySkill);
            command.Parameters.AddWithValue("@last_position", player.transform.position.ToString());
            command.Parameters.AddWithValue("@last_login", DateTime.UtcNow);
            
            command.ExecuteNonQuery();
        }
    }
}

Integration Steps

The following steps outline the process of integrating the uMMORPG database with HomeSalad's unique systems:

1. Database Schema Extension

The first step was to extend the uMMORPG database schema to include tables for HomeSalad's unique features, such as plants, environment states, and relationships.

// Example of schema extension
public void ExtendDatabaseSchema()
{
    using (var connection = new SqliteConnection(connectionString))
    {
        connection.Open();
        
        using (var command = connection.CreateCommand())
        {
            // Create environment table
            command.CommandText = @"
                CREATE TABLE IF NOT EXISTS environment (
                                string building_type
                                int level
                                int durability
                                vector3 position
                                quaternion rotation
                                datetime build_time
                            }
                            
                            relationships {
                                int id PK
                                int character_id FK
                                string npc_name
                                int relationship_value
                                int gift_count
                                datetime last_interaction
                            }
