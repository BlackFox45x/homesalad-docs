Skill Tree System

This section details HomeSalad's skill tree system, showing how players can develop and specialize their character's abilities through branching progression paths.

Skill Tree Overview

HomeSalad's skill tree system provides a visual representation of character development paths, allowing players to plan their progression and specialization. Unlike traditional RPG skill trees that use abstract point allocation, HomeSalad's skill trees unlock through practical application and experience.

AI Prompt: "A comprehensive skill tree diagram for a farming/community game showing five main branches: Cultivation (green), Construction (brown), Community (blue), Conservation (teal), and Crafts (orange). Each branch has 3-4 sub-branches with 5-6 nodes each. Clean, modern design with connecting lines showing prerequisites."

Skill Tree Structure

Each of the five skill categories has its own skill tree with multiple specialization branches:

Each path has 5-6 progressive skills with connecting lines showing prerequisites. Green color scheme with plant-themed icons for each skill."

Crop Farming Branch

  • Basic Crop Tending (Level 1)

    Ability to grow basic crops with standard care techniques

    Unlocks: Basic vegetable crops, simple watering techniques

  • Crop Rotation (Level 2)

    Understanding of crop families and rotation benefits

    Unlocks: 10% yield increase, reduced pest problems

  • Intensive Planting (Level 3)

    Techniques for maximizing space through companion planting

    Unlocks: 25% more plants per garden bed, companion benefits

  • Season Extension (Level 4)

    Methods to extend growing seasons with protection

    Unlocks: Cold frames, row covers, early/late planting

  • Master Crop Production (Level 5)

    Advanced techniques for maximum yield and quality

    Unlocks: 50% yield increase, premium quality crops, teaching ability

Permaculture Branch

  • Permaculture Basics (Level 1)

    Understanding of core permaculture principles

    Unlocks: Basic guild planting, water management awareness

  • Guild Design (Level 2)

    Creating plant communities that support each other

    Unlocks: 15% reduction in maintenance, pest resistance

  • Food Forest Layers (Level 3)

    Understanding vertical space utilization in planting

    Unlocks: Canopy, understory, shrub, herb, ground cover, root, and vine layers

  • Ecological Design (Level 4)

    Creating self-sustaining garden ecosystems

    Unlocks: 30% water reduction, 40% maintenance reduction

  • Permaculture Mastery (Level 5)

    Advanced integration of all permaculture systems

    Unlocks: Self-sustaining gardens, maximum biodiversity, teaching ability

Plant Breeding Branch

  • Seed Saving (Level 1)

    Basic techniques for harvesting and storing seeds

    Unlocks: Ability to save seeds from non-hybrid plants

  • Selection Techniques (Level 2)

    Identifying and selecting for desirable traits

    Unlocks: 10% improved seed quality, trait selection

  • Cross-Pollination (Level 3)

    Controlled breeding between compatible plants

    Unlocks: Ability to create crosses, new variety development

  • Variety Stabilization (Level 4)

    Techniques to stabilize new crosses into true-breeding varieties

    Unlocks: Creation of stable varieties, 25% trait improvement

  • Master Plant Breeder (Level 5)

    Advanced breeding for multiple traits and local adaptation

    Unlocks: Unique varieties, 50% improvement in selected traits, teaching ability

Orchard Management Branch

  • Fruit Tree Basics (Level 1)

    Fundamental care of fruit trees and bushes

    Unlocks: Basic fruit tree planting and care

  • Pruning Techniques (Level 2)

    Proper pruning for tree health and production

    Unlocks: 15% increased fruit production, improved tree form

  • Grafting (Level 3)

    Joining different varieties to a single rootstock

    Unlocks: Multi-variety trees, propagation of rare varieties

  • Integrated Orchard Systems (Level 4)

    Combining trees with complementary plants

    Unlocks: 30% pest reduction, 25% yield increase

  • Orchard Mastery (Level 5)

    Advanced orchard design and management

    Unlocks: Maximum fruit quality and yield, extended harvest seasons, teaching ability

Skill Progression Mechanics

Skills in HomeSalad progress through practical application rather than abstract point allocation:

Experience Acquisition

  • Practice: Performing related activities provides skill experience
  • Quality Bonus: Higher quality results provide more experience
  • Complexity Bonus: More challenging applications provide more experience
  • Innovation Bonus: First-time or creative applications provide bonus experience
  • Teaching Bonus: Sharing knowledge with others provides experience

Skill Level Thresholds

Level Experience Required Capabilities
1 (Novice) 0 XP Basic functionality, high failure rate, limited efficiency
2 (Apprentice) 100 XP Reliable basic techniques, moderate efficiency, occasional failures
3 (Practitioner) 300 XP Advanced techniques, good efficiency, rare failures
4 (Expert) 700 XP Specialized techniques, excellent efficiency, teaching capability
5 (Master) 1500 XP Complete mastery, maximum efficiency, innovation capability

Specialization Choices

As players progress, they face meaningful choices about specialization:

  • Branch Selection: Players can focus on specific branches or spread across multiple
  • Time Investment: Reaching mastery in one area limits time available for others
  • Complementary Skills: Some skill combinations provide special synergy bonuses
  • Community Role: Specialization defines a player's role in the community
  • Teaching: Masters can accelerate others' learning in their specialty

Skill Tree Interface

The skill tree is accessed through the player's journal and provides several features:

AI Prompt: "A clean, modern UI design for a skill tree interface in a farming/community game. The interface shows a selected skill with details panel on the right displaying prerequisites, benefits, and progress. The left side shows a navigable tree with color-coded branches and nodes. Include progress bars, unlock indicators, and a natural paper/journal aesthetic."

The technical implementation of the skill tree interface is detailed in the UI MCP Implementation documentation, which explains how the skill system integrates with the UI architecture and player progression tracking.

Interface Features

  • Visual Progression: Clear visual indicators of current level and progress
  • Skill Details: Detailed information about each skill and its benefits
  • Prerequisites: Clear indication of what's needed to unlock each skill
  • Recommended Activities: Suggestions for activities to advance each skill
  • Mastery Tracking: Overall progress toward mastery in each category

Design Note: The skill tree system is designed to encourage organic character development through gameplay rather than abstract point allocation. Players naturally develop skills in areas they enjoy and use frequently, while still having visibility into potential specialization paths.
